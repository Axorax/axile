You can actually change where the (edited) symbol shows when you edit a message. This trick has been around for quite a while now. Here is what a normal edited message should look like:

Hello world (edited)

But by using this trick, you can make the (edited) symbol appear anywhere. For example:

Hello(edited) world

With using this trick, you can do some cool stuff. Like changing the word 'edited' to the edited symbol.

I (edited) this message!

(edited) Message Trick
Step 1/3: Send any message
At first, you have to send a message anywhere. It works everywhere including; voice channels, DMs, etc.

Axile is cool!
Step 2/3: Click edit and paste the special character
Secondly, you have to click edit. Then you need to copy and paste the special character where you want the (edited) character to appear. The special character is known as U+202B. You can copy it from this site. Or, click the below button to copy it.



Axile(edited) !is cool

The 202B is invisible. It may appear that you pasted nothing. It is hard to work with. You should paste it and avoid editing. In the above, the exclamation(!) symbol has appeared before `is`. That is not a typo. That's just how 202B works. I don't know if you can prevent it or not.
Step 3/3: Save your changes
Now all you have to do is save your edit and your done!

The edit is visible for everybody. The character is a unicode punctuation character that aligns embed content from right-to-left. Discord may patch this in the future.
