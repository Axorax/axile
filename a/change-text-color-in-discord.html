You can change the color of your Discord text by using a multi-line code block. To use a code block you have to use the backtick(`) character. In this article, we will show you how to use code blocks to get different colored texts in Discord.
Plain Grey
To get this color, all you have to do is just type some text in a code block.
```
axile is a cool site!
```
axile is a cool site!
Light Green
Type diff after the first three backticks and use a + symbol in the start of every line.
```diff
+ axile is a cool site!
```
